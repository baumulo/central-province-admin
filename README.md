# Central Province Administration Hub

Official information sharing Progressive Web App (PWA) for senior officers and Heads of Departments in Central Province, Zambia.

## 📱 About the App

This mobile-friendly web application enables efficient communication and information sharing among provincial administration officers in Central Province. The app provides a centralized platform for official announcements, meeting notifications, and departmental updates.

## 🏛️ For Central Province Officials

**Location:** Kabwe, Central Province, Zambia  
**Target Users:** Senior Officers, Heads of Departments (HODs), Provincial Administration Staff

## ✨ Key Features

- **📢 Official Announcements** - Share important updates and notices
- **📅 Meeting Notifications** - Schedule and announce meetings
- **📊 Priority System** - High, medium, and low priority classifications
- **📍 Location Tags** - Kabwe and Central Province location tracking
- **📱 Mobile-First Design** - Optimized for smartphones and tablets
- **🔔 Real-time Updates** - Stay informed with latest information
- **👥 User Profiles** - Officer information and contact details

## 💾 Installation Instructions

### For Android Users:
1. Open the app URL in Chrome browser
2. Look for the "📱 Install App" button
3. Tap "Install" or use Chrome menu → "Add to Home Screen"
4. App will appear on your home screen

### For iPhone Users:
1. Open the app URL in Safari browser
2. Tap the Share button (square with arrow up)
3. Select "Add to Home Screen"
4. Tap "Add" to install

### For Desktop Users:
1. Open the app URL in Chrome, Edge, or Firefox
2. Look for the install prompt or browser menu option
3. Click "Install" to add to your desktop

## 🚀 Getting Started

### First Time Setup:
1. **Access the App** - Visit the provided URL
2. **Install on Device** - Follow installation instructions above
3. **Create Posts** - Use the blue "+" button to share information
4. **Set Priority** - Choose High, Medium, or Low priority for posts
5. **Select Category** - Categorize posts (Meeting, Workshop, Report, etc.)

### Daily Usage:
- **View Updates** - Check the home feed for latest information
- **Create Posts** - Share official announcements and notices
- **Check Notifications** - Stay updated with unread notifications
- **Update Profile** - Keep contact information current

## 📋 Post Categories

- **General** - Standard announcements
- **Meeting** - Meeting notifications and schedules
- **Workshop** - Training and workshop announcements
- **Report** - Official reports and documentation
